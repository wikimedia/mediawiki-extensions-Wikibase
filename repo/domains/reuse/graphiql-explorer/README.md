# Wikibase GraphQL Explorer

Wikibase GraphQL Explorer is using [GraphiQL](https://github.com/graphql/graphiql/blob/main/packages/graphiql/README.md).

## Setup

* `npm i` to install dependencies
* configure the `VITE_GQL_ENDPOINT_URL` environment variable to the GraphQL endpoint URL to use
* `npm run dev` to start the dev server
* `npm run build` to build production ready transpiled files. Find the output in `Wikibase/docs/graphql-explorer/` folder.
